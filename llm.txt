# PandaLatam - Chinese Learning Platform

> **PandaLatam** is a high-performance educational platform engineered to bridge the linguistic gap for Spanish speakers learning Chinese.

## Core Stack
- **Frontend**: HTML, Vanilla JS, Tailwind CSS, Eleventy (11ty) SSG
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **AI & Security**: Groq SDK, JWT, Google OAuth2, Helmet, Express-Rate-Limit

## Key Features
- **Dynamic Lessons**: Content structure categorized by levels (A1 to C1).
- **Word of the Day**: Interactive widget powered by AI (Groq / LLaMA) providing daily vocabulary and quizzes.
- **AI Chat Assistant**: Integrated conversational bot for practicing Chinese.
- **Lesson Editor & Community**: Users can contribute HTML/Rich-text lessons that administrators can approve or reject in the dashboard.

## Development Commands
- `npm run dev`: Starts the Eleventy watcher, Tailwind compilation, and Node/Express server concurrently.
- `npm run build`: Generates the production build into the `_site` directory.
- `npm run start:prod`: Starts the server via PM2 for production use.
